# rust_tracer

### Ray tracer written in rust.


Follows ["Ray Tracing in one weekend"](https://raytracing.github.io/) book by Peter Shirley. Code here is my implementation in Rust.

-  [x] Multithreading
-  [x] Spheres
-  [x] Real Time Preview of Rendering (uses Piston)
-  [x] Progress Logging
-  [x] Movable Camera
-  [x] Depth of Field
- [ ] Triangles
- [ ] Motion Blur
- [ ] Bounding Volumes
- [ ] Solid Textures
- [ ] Lights
- [ ] Density Volumes
